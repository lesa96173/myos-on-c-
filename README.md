как запустить проект:

гайд для винды(для линукс не знаю, я бездарь на поводке у корпараций)

0.Скачайте сурс проекта

1.Устанавливаем wsl, если вы на винде

2.Качаем дистрибутив ububntu (не спрашивайте как, смотрите в инете)

3.Прописываем sudo apt update && sudo apt install nasm gcc g++ make qemu-system-x86 -y

4.Переходим в директорию через cd "путь к файлу"

5.Прописываем make clean

6.Прописываем make run

Все! Пользуйтесь не заляпайтесь.
